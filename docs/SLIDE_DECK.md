# CipherLink Slide Deck (10 Slides)

## 1) Title
- CipherLink: Multi-Tier SCF Fraud Intelligence
- Tagline: Stop Fraud Before Disbursement
- Team name + event

## 2) Problem
- SCF fraud is growing in complex multi-tier chains
- Invoice-level checks miss network-level abuse
- Losses hit lenders, suppliers, and trust

## 3) Real Case
- Highlight the $47M phantom invoice scenario
- Emphasize "individually valid, collectively fraudulent"

## 4) Why Existing Systems Fail
- Siloed checks per invoice
- No cross-relationship graph visibility
- Late detection after funds leave

## 5) Our Solution
- CipherLink decision engine
- Ingest → fingerprint → graph analysis → risk score
- Explainable output for fast action

## 6) System Architecture
- FastAPI + PostgreSQL backbone
- NetworkX graph analytics
- Risk engine + explanation layer
- React dashboard for operations

## 7) Key Features
- Duplicate detection
- Cycle detection
- Velocity/high-frequency/hub anomalies
- Explainable risk scoring (LOW/MEDIUM/HIGH)

## 8) Demo Flow
- Upload invoice
- Duplicate blocked
- Graph updates
- High-risk alert and reason panel

## 9) Impact
- Pre-disbursement fraud prevention
- Faster decision confidence for banks
- Better transparency across supply chain tiers

## 10) Future Scope
- Dynamic NFT invoice identity
- Zero-knowledge privacy proofs
- GNN model for adaptive fraud scoring
- Cross-lender shared risk intelligence
